Terdapat tiga model yaitu:

1. Member
   - \_id (string)
   - name (string)
   - phone (string)
   - address (string)
   - profile_picture (string)
   - username (string)
   - password (string)
   - role (string)

2. Buku
   - \_id (string)
   - title (string)
   - isbn (string)
   - pages (string)
   - genres (string)
   - cover (string)
   - description (string)
   - author (string)
   - publisher (string)
   - stock (string)
   - location (string)

3. Peminjaman
   - \_id (string)
   - book_id (string)
   - user_id (string)
   - tanggal_peminjaman (date)
   - tanggal_pengembalian (date)
